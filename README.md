# Hebbian-Consolidation-during-Sleep-Drives-the-Semantization-of-Episodic-Sequences
This repository contains all the code and data related to the article currently under review, “Hebbian Consolidation during Sleep Drives the Semantization of Episodic Sequences: A Neurocomputational Model”. Ursino, Pirazzini, and Valgimigli; Scientific Reports, 2026 (UNDER REVIEW)

GENERAL README FILE

This repository contains seven different folders, each containing codes for simulating the memory model described in the article 'Hebbian Consolidation during Sleep Drives the Semantization of Episodic Sequences: A Neurocomputational Model: Ursino, Pirazzini, and Valgimigli. Scientific Reports, 2026. UNDER REVIEW). 

Each folder contains all the materials needed to view simulations of all the figures in the article.

More details are contained in the comments to the lines of code in the various scripts. The model was implemented exclusively in the MATLAB environment.

For any doubts you can contact the corresponding author Mauro Ursino at mauro.ursino@unibo.it
